# Digispark-ATTiny85

![Autor: Max Muller](https://img.shields.io/badge/Autor-Max%20Muller-blue?style=for-the-badge&logo=github)
![Digispark ATTiny85](https://img.shields.io/badge/ATTiny85-Microcontrolador-orange?style=for-the-badge&logo=arduino)

Repositório dedicado a projetos educacionais desenvolvidos com o microcontrolador **Digispark ATTiny85**, explorando sua capacidade de simular dispositivos USB e automatizar tarefas.  
Este projeto tem foco em demonstrar lógica aplicada, controle de hardware e implementação de rotinas embarcadas, alinhado ao uso real em automação e experimentação eletrônica.

## Objetivo do Projeto
Apresentar aplicações práticas do ATTiny85 para fins acadêmicos, ilustrando como um dispositivo compacto pode executar rotinas USB simuladas com eficiência e baixo custo.

## Estrutura
O código-fonte está localizado em arquivos separados dentro do repositório, mantendo organização limpa e pronta para consulta de estudantes e recrutadores.
